Introduction

The aim of the project is to make a website that will depict some data visualizations from the Trento tree dataset. Dataset carries information about trees in the city Trento. For making a website we will use html, css and D3.js which is javascript library for creating dynamic data visualizations. With help of D3.js we will analyze which type of graph will be more suitable for the Trento tree dataset.
